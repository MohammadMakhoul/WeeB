# 🌍 WEEB - Travel Landing Page

WEEB is a responsive travel landing page built from scratch using HTML, CSS, and JavaScript.

The project was created to practice front-end development fundamentals, focusing on clean design, responsive layouts, and user-friendly navigation.

## 🚀 Live Demo

https://mohammadmakhoul.github.io/WeeB/

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript 

## ✨ Features

- Responsive design for different screen sizes
- Modern and clean user interface
- Travel destination showcase
- Interactive navigation
- Smooth user experience
- Optimized layout structure

## 📚 What I Learned

Through building this project, I improved my understanding of:

- Front-end development fundamentals
- Responsive web design
- CSS layouts and styling
- JavaScript 
- Debugging and problem-solving


## 🎯 Purpose

The goal of this project was to strengthen my web development skills by building a complete landing page from scratch without relying on frameworks.
